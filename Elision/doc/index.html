Elision

Welcome to the home page for the Elision rewriter.

Elision is a term rewriting library and REPL.  It is under construction, so set your expectations accordingly.  Right now it is certainly not stable and not recommended for use in any serious applications.

The rewriter is undergoing heavy development, and we hope to have a full release of it soon.

More information is available from the links below.

Note that to build or run Elision you must have Scala installed.  To build Elision you must have Apache Ant installed.

To use the runnable jar, execute it with the scala command:
scala elision-@DSTAMP@.jar
